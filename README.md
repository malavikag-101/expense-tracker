# expense-tracker

The Expense Tracker Application is a simple web-based application developed using Spring Boot, Thymeleaf, and MySQL.This project helps users record and manage their daily expenses by allowing them to add, view, and delete expense entries.It also calculates the total expense amount, making it easy to track spending habits.
This project was built to understand the basic working flow of a Spring Boot MVC application, including database integration, CRUD operations, and server-side rendering using Thymeleaf.

Features
Add new expenses (amount, category, date)
View all expenses in a table format
Delete expenses
Automatically calculate total expense amount
Simple and user-friendly interface

Technologies Used
Java
Spring Boot
Spring Data JPA
Thymeleaf
MySQL
Maven

Project Structure
expense-tracker
 ├── controller
 ├── service
 ├── repository
 ├── model
 ├── templates
 │   ├── add.html
 │   └── list.html
 └── application.properties

How to Run the Project
Clone the repository
Create a MySQL database named expense_db
Update database credentials in application.properties
Run the project as Spring Boot App
